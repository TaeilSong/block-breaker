# Block Breaker Bot Lab

Canvas 기반 블록격파 웹앱입니다. 표준적인 블록격파 규칙에 맞춰 벽 쿠션, 패들 반사각, 행별 점수 계산을 구현했고, 같은 시드로 재현 가능한 자동 플레이 봇과 벤치마크를 포함합니다.

## Features

- 좌우 벽과 천장 쿠션 반사
- 바닥 낙하 시 생명 감소
- 상단 2줄 7점, 다음 2줄 5점, 다음 2줄 3점, 하단 2줄 1점
- 패들 타격 위치에 따른 반사각 변화
- 자동 플레이 봇
- 고정 타임스텝과 시드 기반 봇 성능 측정
- Node 내장 테스트 러너 기반 엔진 테스트

## Run

```bash
npm install
npm start
```

브라우저에서 표시된 로컬 주소를 열면 게임을 플레이할 수 있습니다.

## Controls

- `←` / `→` 또는 `A` / `D`: 패들 이동
- `Space`: 서브
- `봇 플레이`: 자동 플레이 토글
- `측정`: 같은 시드로 봇 성능 측정

## Test

```bash
npm test
```

## Benchmark

```bash
npm run benchmark -- 20260706 12
```

첫 번째 인자는 시작 시드, 두 번째 인자는 실행할 게임 수입니다. 같은 시드와 게임 수를 사용하면 동일한 결과를 재현할 수 있습니다.
